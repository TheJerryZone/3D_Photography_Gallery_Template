3D Photography Gallery Template

A modern **3D photo gallery website template** built with **React**, **Next.js**, and **TailwindCSS**, featuring an interactive 3D slideshow of images. Ideal for photographers, portfolios, and creative showcases.

---

## Features

1. 3D scroll-based image gallery  
2. Smooth and interactive experience  
3. High performance with React & WebGL  
4. Fully responsive layout  
5. Ready-to-use template structure

---

## Installation

### Prerequisites

- Node.js (v14+)  
- npm 

### Steps

1.  Clone the repository:
       git clone = https://github.com/TheJerryZone/3D_Photography_Gallery_Template.git
2.  Navigate into the project folder = cd 3D_Photography_Gallery_Template
3.  Install dependencies = npm install 
4.  Start the development server: npm run dev

Open http://localhost:3000 to view in your browser.
